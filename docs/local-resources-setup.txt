# Configuración de Recursos Locales para el Proyecto

Este archivo te guiará en el proceso de descargar y configurar todos los recursos externos que utiliza el proyecto para que funcionen de manera local sin conexión a internet.

## 1. Estructura de Carpetas Recomendada

Crea la siguiente estructura de carpetas en la raíz de tu proyecto:

```
vendor/
  ├── bootstrap-icons/
  │   ├── fonts/
  │   └── bootstrap-icons.min.css
  ├── font-awesome/
  │   ├── css/
  │   └── webfonts/
  ├── animate-css/
  │   └── animate.min.css
  └── fonts/
      └── inter/
          └── inter.css
          └── (archivos de fuentes)
```

## 2. Bootstrap Icons (v1.11.3)

### Descargar:
1. Visita: https://github.com/twbs/icons/releases/tag/v1.11.3
2. Descarga el archivo ZIP "bootstrap-icons-1.11.3.zip"
3. Extrae el contenido

### Instalar:
1. Copia la carpeta `font` del ZIP a `vendor/bootstrap-icons/fonts`
2. Copia el archivo `bootstrap-icons.css` a `vendor/bootstrap-icons/bootstrap-icons.min.css`

## 3. Font Awesome (v6.5.1)

### Descargar:
1. Visita: https://fontawesome.com/download
2. Descarga el archivo "Free for Web"
3. Extrae el contenido

### Instalar:
1. Copia la carpeta `css` del ZIP a `vendor/font-awesome/css`
2. Copia la carpeta `webfonts` del ZIP a `vendor/font-awesome/webfonts`

## 4. Animate.css (v4.1.1)

### Descargar:
1. Visita: https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css
2. Guarda el archivo como `vendor/animate-css/animate.min.css`

## 5. Google Fonts (Inter)

### Descargar:
1. Visita: https://fonts.google.com/specimen/Inter
2. Descarga las fuentes (weights: 400, 500, 600, 700)
3. O usa Google Webfonts Helper: https://gwfh.mranftl.com/fonts/inter

### Instalar:
1. Crea una carpeta `vendor/fonts/inter`
2. Coloca las fuentes descargadas en esta carpeta
3. Crea un archivo `inter.css` en la misma carpeta con el siguiente contenido:

```css
/* inter-regular - latin */
@font-face {
  font-family: 'Inter';
  font-style: normal;
  font-weight: 400;
  src: url('./inter-regular.woff2') format('woff2'),
       url('./inter-regular.woff') format('woff');
}
/* inter-500 - latin */
@font-face {
  font-family: 'Inter';
  font-style: normal;
  font-weight: 500;
  src: url('./inter-500.woff2') format('woff2'),
       url('./inter-500.woff') format('woff');
}
/* inter-600 - latin */
@font-face {
  font-family: 'Inter';
  font-style: normal;
  font-weight: 600;
  src: url('./inter-600.woff2') format('woff2'),
       url('./inter-600.woff') format('woff');
}
/* inter-700 - latin */
@font-face {
  font-family: 'Inter';
  font-style: normal;
  font-weight: 700;
  src: url('./inter-700.woff2') format('woff2'),
       url('./inter-700.woff') format('woff');
}
```

## 6. Modificar los archivos HTML

Ahora, necesitas modificar los archivos HTML para que utilicen los recursos locales:

### En index.html:

Reemplaza:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css">
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css" />
```

Con:
```html
<link href="vendor/fonts/inter/inter.css" rel="stylesheet">
<link rel="stylesheet" href="vendor/bootstrap-icons/bootstrap-icons.min.css">
<link rel="stylesheet" href="vendor/font-awesome/css/all.min.css">
<link rel="stylesheet" href="vendor/animate-css/animate.min.css">
```

### En login.html:

Reemplaza:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

Con:
```html
<link href="vendor/fonts/inter/inter.css" rel="stylesheet">
```

## 7. Verificar Rutas Bootstrap

También asegúrate de que los archivos de Bootstrap estén disponibles localmente:

```html
<link href="../bootstrap/css/bootstrap.min.css" rel="stylesheet">
<script src="../bootstrap/js/bootstrap.bundle.min.js"></script>
```

Si estos archivos no existen en estas rutas relativas, descarga Bootstrap (v5.x) desde https://getbootstrap.com/docs/5.3/getting-started/download/ y colócalos en la ubicación correspondiente.
